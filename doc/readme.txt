UTTERANCE PLANNER FOR DFKI COGX (DORA & GEORGE), NIFTI & ALIZ-E
-------------------------------------------------------------
VERSION: 2010-04-03

AUTHOR:
- Bernd Kiefer (kiefer@dfki.de)

SVN REPOSITORY: http://subversion.dfki.de/talkingrobots/branches/utteranceplanner

--------------------------------------------------------
REQUIREMENTS:
- Java (tested on version 1.6.0_15)
- Ant (tested on version 1.7.1)

INSTALLATION INSTRUCTIONS:
1. Checkout the SVN repository to a new folder.
2. Run 'ant in the directory' 'utteranceplanner/uttplanner/'.
3. Run the utterance planner debugger by executing the script 'updebugger' in the directory 'utteranceplanner/uttplanner/' (make sure it is executable, e.g. run 'chmod a+x utteranceplanner/uttplanner/updebugger').
